8bit-clocks
===========


Dies ist das Ergebnis des [8bit] Seminares bei Jens Ewald.
Die Seminarteilnehmer haben Uhren in Form von Klassen geschrieben 
und diese in eine Hauptskizze implementiert. 

Ziel der Abschlussübung ist es, 
kollaborativ über Git an einem Projekt zu arbeiten.
